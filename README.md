# LibertyCode Loading screen
### It's just a simple, clean looking loading screen :)
## **Configuration**
`nui/style.css`
```css
:root {
    --accent-color: #e9e114; 
    --background: #000;
    --light-text: #aaa;
    --loading-bg: #111;
    --transition: 1s;
}
```
To change logo, just replace `nui/logo.png` with your logo.
